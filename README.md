
# Round the Home Tempreture Sensor(s). 

Quick hack for temp sensors around the home

Sends basic data to pub/sub data mtqq.

| NodeMCU (ESP8266) |

### Wiring

| BMP280 | NodeMCU |
| ------ | ------ |
| VCC | VCC or 3.3v |
| GND | GND |
| SCL | D2 |
| SDA | D1|
